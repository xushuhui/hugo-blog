## 方法和属性的可见性修饰符

- 可见性修饰符用在类、成员方法、构造方法、静态方法和属性上，其可见性
的范围是一样的
- 看代码，学习可见性修饰符：
    - public：全局可见
    - 缺省：当前包可以见
    - private：当前类可以见
- 理解访问修饰符：不只是为了限制不让人用，更为了有规矩才成方圆。成员变量应该是 private 的，不需要让外部使用的方法应该都是 private 的
